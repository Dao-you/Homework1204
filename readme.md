# kotlin 程式碼練習

## 心得

這次老師讓我們自己回家學了 Fragrement，這個東西本來不太熟悉，但是後來發現其實跟之前學的 Activity 也差不多，一樣有一個 Class，有一個獨立的 XML 排版，有其他的行為等，只是必須放在 Activity 底下。
透過 Fragement，就可以做到更多的小細節。

另外也跟之前一樣把上課做的 LAB 用 Kotlin 再寫一次了，隨著程式的複雜度越來越高，兩者的差異也越來越大了，雖然都還是遵循著老師上課界式的物件導向，各種功能的名稱也都相差無幾，讀是一定可以讀懂的，如果要寫的話可能就會比較卡，目前我還是比較習慣 Java 的寫法，但是之後有匙間好像還是應該來把 Kotlin 從頭學過會比較好。
